HOW IT WORKS:

Example 1: NetworkManager-managed WiFi
  User Action: Click "Connect" on wlp3s0
  Detection:   Interface managed by NetworkManager
  Backend:     Uses nmcli device connect wlp3s0
  Result:      ✓ Connected via networkmanager

Example 2: Unmanaged Ethernet
  User Action: Click "Connect" on eth0
  Detection:   Interface not managed
  Backend:     Uses dhcpcd eth0
  Result:      ✓ Connected via DHCP client

Example 3: Static IP on systemd-networkd
  User Action: Set static IP on eth1
  Detection:   Interface managed by systemd-networkd
  Backend:     Creates /etc/systemd/network/50-eth1.network
  Result:      ✓ Static IP configured via systemd-networkd
